The entire integration

Two tags plus one script — paste into any HTML page.

<od-offer-grid
  publication="blue-gr-summer"
  api-base="https://odisea-kw3x.onrender.com"
  api-key="od_pub_blue_horizon_demo_key_v1"
  columns="3"></od-offer-grid>

<script src="https://odisea-kw3x.onrender.com/components/loader.js"
        data-version="1"></script>

The api-key is a publishable key: read-only, tied to the agency, revocable, and lockable to specific domains. Without a valid key the endpoints return 401 — no key, no offers.
